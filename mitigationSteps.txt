Immediate actions guidance
- Delete node_modules and lockfiles having malicious versions
- Clean npm cache
- do a dry-run to check what packages will be installed (npm i --dry-run --ignore-scripts)
- reinstall safe packages, only using npm install with --ignore-scripts flag
- pin to known-good versions using package-lock.json and use (npm ci --ignore-scripts) for further installs

See also: https://github.com/M-S/npmSafeCheck?tab=readme-ov-file#immediate-actions-guidance
